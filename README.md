# Paradise Nursery – E-Plant Shopping Application

Paradise Nursery is a React-based e-plant shopping application developed as part of a Coursera assessment.

## Project Overview
This application allows users to browse different categories of plants, add plants to a shopping cart, update quantities, and view the total cost dynamically.

## Features
- Landing page with background image
- Product listing with multiple plant categories
- Shopping cart with quantity management
- Redux Toolkit for state management

## Technologies Used
- React
- Redux Toolkit
- JavaScript
- CSS

## Usage
Click on “Get Started” to browse plants and add them to the cart.
